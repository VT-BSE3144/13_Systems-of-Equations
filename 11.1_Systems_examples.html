Examples

Imagine you have a simple set of equations, and you plot them out. The solution is at the intersection if you plot them. Let’s solve using matrix algebra:

Example 1 plot

We’ll employ the solve function. When you enter just the A matrix, you get the inverse as the output. When you enter the A and B matrix/vectors, you’ll get the x vector.

%*% = matrix multiplication operator solve = function for solving systems, can input A or A & B.

Simple Matrix and solving for X

A <- matrix(c(3, 2, -1, 2), ncol = 2, byrow=TRUE)
Ainv <- solve(A)
B <- matrix(c(18,2), ncol = 1, byrow=TRUE)
X <- Ainv %*%B
#or 
X <- solve(A,B)
X
##      [,1]
## [1,]    4
## [2,]    3

To find effect of reactor 3 on reactor 1….

This is where the matrix inverse is really cool, and provides an opportunity to quantify change in one part of a system on another component. Here, imagine we make a change in reactor 3, and want to know how this in turn changes the concentraiton in reactor 1. We’ll walk through this in class in some more detail, but the basic premise is the following: when we multiple the inverseA by b, the first equation for reactor 1 is the following:

Equation

Thus we can use the circled part to determine the effect of reactor 3 on 1!

x1effect <- invA[1,3] * b[3]
x1effect
